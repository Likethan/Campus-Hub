# CampusHub

> **One Platform. Every Student Need.**

> **Note:** This is a condensed starter README due to generation limits.

## Overview

CampusHub is a premium Student Life Management Platform built with React
19, TypeScript, Vite and Tailwind CSS v4.

## Features

-   Dashboard
-   Attendance
-   Assignments
-   Placements
-   Events
-   Clubs
-   Notes
-   Student Profile
-   Campus AI Assistant
-   Dark Mode
-   Responsive UI

## Tech Stack

  Layer       Technology
  ----------- -----------------------
  Frontend    React 19
  Language    TypeScript
  Build       Vite
  Styling     Tailwind CSS v4
  Animation   Framer Motion
  Charts      Recharts
  Routing     React Router v7
  Forms       React Hook Form + Zod
